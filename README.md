# Daily Calendar (Himekuri)

A simple, premium-style daily calendar web application built with Vite and Vanilla JavaScript.

## Features
- Displays the current date, weekday, and Rokuyo.
- "Tear off" animation to reveal the next day.
- Daily inspirational quotes.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```
